# 输出要求

严格按以下 JSON 格式输出，未找到的项不要出现：

```json
{
  "datapoints": [
    {
      "canonical_key": "total_assets",
      "name_zh": "资产总计",
      "name_en": "Total assets",
      "value": 1234567.89,
      "value_text": "1,234,567.89",
      "unit": "人民币百万元",
      "currency": "CNY",
      "period": "2024-12-31",
      "evidence": {
        "page": 45,
        "bbox": [56.7, 234.1, 540.3, 251.0],
        "snippet": "资产总计 1,234,567.89",
        "section": "合并资产负债表"
      },
      "confidence": 0.95
    }
  ]
}
```

注意：
- value 必须是 float 类型，去除千分位逗号和单位
- 若片段含"百万元"/"千元"等单位，要忠实记录在 unit 字段，不要擅自换算到元
- evidence.snippet 不超过 100 字
- 没有把握的项目 confidence < 0.7，避免胡编乱造
