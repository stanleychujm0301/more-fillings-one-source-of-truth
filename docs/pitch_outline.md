## 设计要点（执行 P7 时注意）

- 每页字数控制在 30 字以内，**演示靠口播不靠 PPT 字**
- 主蓝 #00338D 用在大标题、强调元素；浅蓝 #0091DA 用在副元素
- 不要用 emoji
- 数据用大字号 + 衬色块（参考 `optimize_gt_ht_ppt.py` 中已有的样式）
- 截图证据请用 PNG，分辨率 ≥ 2K（路演大屏需要）
- 最后一页二维码生成可用 `qrcode` Python 库
