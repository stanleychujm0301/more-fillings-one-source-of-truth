## R1 — LLM 准则推理产生幻觉

- **概率**：高
- **影响**：高（直接关系到亮点 1 可信度）
- **触发条件**：检索到的准则条款相似度低 / LLM 编造引用
- **兜底措施**：
  1. RAG 强制返回引用源（StandardCitation 含 source 字段）
  2. UI 标"AI 建议"非"AI 结论"，需要审计师覆盖确认
  3. confidence < 0.85 直接降级为"待人工复核"
- **负责人**：P4
- **状态**：开放

## 风险评估方法

每条风险按"概率 × 影响"四宫格分类：
- **致命**：必须兜底（R5）
- **重大**：需要明确缓解（R1 R2 R3 R7 R10）
- **次要**：监控不阻塞（R4 R6 R9）
- **低危**：备案（R8）
